DANIEL HAIM

Castle Crusher


For one of my computer science projects, I collaborated with Mark Lindblad to develop a retro dungeon style interactive game. The goal of the game is to catch the ghost characters that randomly walk around the world before the time is up.

The game's map is built using random rooms and hallways generator. The player can select from varying levels of difficulties and save or load new and old games. To navigate the map the player uses the keyboard and in addition, the player can also toggle a masked view of the map for an extra challenge or rotate the map 90 degrees left or right. When the player hovers over the different tiles in the map the user can see a description of each tile.

We used Java to code the games engine, and IntelliJ IDEA and GitHub to collaborate and test the games.
